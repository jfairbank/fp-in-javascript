    Functional Programming Basics in JavaScript

  • ¶

    This website aims to provide a one stop source for teaching functional programming paradigms by implementing them in JavaScript — ES2015 to be exact! If you have wondered how to implement a recursive map function or create a curry function, then this website will explain it to you.

    Functional programming continues to grow in popularity and is becoming increasingly relevant in the world of JavaScript frameworks and libraries. Utilize this website to gain better understanding of the foundational concepts of functional programming in order to better appreciate the frameworks and libraries available in JavaScript.

    import { curry } from './function';
import { map } from './array';

const multiply = curry((x, y) => x * y);
const curriedMap = curry(map);
const doubleNumbers = curriedMap(multiply(2));

const numbers = [1, 2, 3];

console.log(doubleNumbers(numbers)); // [2, 4, 6]
